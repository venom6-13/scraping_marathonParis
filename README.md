## Description
Il s'agit de mon premier projet scraping en Python.
Son but étant de scaper le site du classement du marathon de Paris. Je veux faire une communication là-dessus pour que des entreprises de running puissent s'en servir pour des campagnes de pub à destination des utilisateurs finaux ou adapter leurs offres produits/modèles par catégorie.

## Pour le démarrer
Utiliser le fichier 
**main.py**
⚠️ Et utiliser la commande suivante dans le terminal :
```
python3 main.py
```
Pour info, toutes les data scrapées se retrouvent dans le dossier 
**data** => avec le json de base (**50k Coureurs Propres.json**) et le csv à télécharger (**Clean Data.csv**)
